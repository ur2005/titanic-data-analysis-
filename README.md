# Titanic Dataset Analysis

## Topics Covered
- File Operations
- CSV Handling
- JSON Conversion
- NumPy Operations
- Pandas Data Analysis

## Output Files
- survivor.csv
- data.json
- first_class_women.csv

## Technologies Used
- Python
- NumPy
- Pandas
